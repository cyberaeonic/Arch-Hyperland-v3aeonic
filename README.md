# ⚡ Arch-Hyperland-v3aeonic

> A blazing-fast Hyprland dotfiles setup for Arch Linux with Material You dynamic theming.

See the full documentation and screenshots in [`.github/README.md`](.github/README.md).

## Quick Start

```bash
git clone https://github.com/cyberaeonic/Arch-Hyperland-v3aeonic.git
cd Arch-Hyperland-v3aeonic
./setup install
```
